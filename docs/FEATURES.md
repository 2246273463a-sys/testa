# 功能清单（本地优先）

## 已具备

- 三栏工作台（目录 / 列表 / 编辑器）
- 文件夹（树形）与拖拽移动
- 标签：管理、筛选、笔记绑定
- 全局搜索：结果跳转定位
- 编辑器：Markdown 分屏/预览、代码块高亮、代码块复制
- 代办聚合视图：勾选回写、来源跳转
- 思维导图：按 Markdown 标题生成、点击跳转

## 本轮增强（强大功能）

- 入口与交互
  - 目录区显式新建按钮（替代 `prompt`），统一弹窗表单
  - 笔记列表支持多选与批量操作
  - 右键/更多菜单：收藏、置顶、移入回收站、恢复、永久删除

- 内容与组织
  - 收藏（Favorite）与置顶（Pinned）
  - 回收站（Trash）：软删除、恢复、清空
  - 模板（Templates）：将笔记标记为模板；从模板一键新建

- 搜索与筛选
  - 列表过滤：全部/收藏/置顶/回收站/模板
  - 更丰富的筛选条（保持不引起布局抖动）

- 设置（可持续扩展）
  - 外观：动效强度、鼠标悬停光效开关
  - 编辑器：自动保存、换行、字号
  - 数据：导出/导入 JSON 备份（本地“同步/迁移”的基础）

- 代办系统增强
  - 独立代办（不依赖 Markdown 解析）
  - 优先级（低/中/高）与截止日期（Date）

- 知识管理增强
  - 双向链接：`[[标题]]` / `[[标题|别名]]`
  - 反向链接面板 + 缺失笔记一键创建

- 导航与仪表盘
  - 顶栏药丸导航：仪表盘/笔记/代办/导图
  - 仪表盘首页：统计卡片 + 最近编辑

## 后续扩展（需要额外设计/较大工作量）

- 附件/图片上传与管理
- 版本历史与差异对比
- 双向链接与知识图谱（跨笔记引用）
- 账号体系与云同步（或 WebDAV/Git 同步）
- PDF 导出（可先用浏览器打印为 PDF）
